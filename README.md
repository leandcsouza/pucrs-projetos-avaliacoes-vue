# pucrs-frontend

# Projeto Iniciado com a finalidade de ser uma aplicação front-end para testes no curso Desenvolvimento Fullstack PUCRS

## Para iniciar o projeto, inicie o comando abaixo:

```
npm install
```

### Após a instalação, compile o projeto:

```
npm run serve
```
